# ROIC - AI ROI Calculator

## Project Overview

ROIC is a financial-grade AI investment analysis tool that calculates ROI, NPV, IRR, payback period, and risk-adjusted returns to help businesses justify their AI investments.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Navigate to the project directory:
   ```sh
   cd roic
   ```

2. Install the necessary dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Recharts

## Deployment

To deploy the project:

1. Build the production application:
   ```sh
   npm run build
   ```

2. Deploy the contents of the `dist` folder to your preferred hosting provider (e.g., Vercel, Netlify, GitHub Pages).

## Custom Domain

If you have a custom domain, you can point it to your deployment following your hosting provider's instructions.
